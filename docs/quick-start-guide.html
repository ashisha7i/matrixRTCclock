Quick Start Guide

Setting the Time On Your Clock

  1. Plug in your clock to a computer using a USB-C Type Cable.
  2. On your computer, open Google Chrome or a similar browser, open a new tab, and go to: p3d-tetris-clock-setter.surge.sh
  3. Select your timezone from the first drop-down menu.
  4. Click "Connect PS3D Tetris Clock."
  5. A menu will pop up. Select the correct COM port and click Connect.
  6. If you are unsure which port to use:
    • Windows: open Device Manager, open "Ports (COM & LPT)" and note the "COM" number of "USB-SERIAL CH340"
    • Mac: Select "USB Serial" under the dropdown menu.
  7. Please follow to step 8 for Windows, and skip to step 11 for Mac.

WINDOWS

  1. Click "Connect"
  2. Note: If COM3 (or your correct port) does not show up, please unplug and replug the clock while the dropdown is open.
  3. Click "Set Time"
  4. Please proceed to step 16.

MAC

  1. Select "USB Serial"
  2. Note: If USB Serial (or your correct port) does not show up, please unplug and replug the clock while the dropdown is open.
  3. Click "Connect"
  4. Click "Set Time"
  5. Your clock should now reload and display the correct time!
17. You can now plug the clock into any outlet using a USB Type-C Cable and it will retain the correct time.
